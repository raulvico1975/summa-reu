## Regles de priorització

Prioritza en aquest ordre:

1. Home.
2. Pàgines amb major impacte comercial directe.
3. Pàgines que resolen objeccions clau del procés de compra.
4. Pàgines amb valor SEO secundari.
5. Ajustos menors, refinaments visuals o millores no crítiques.

## Criteri d'ús

- No es crea una pàgina nova només perquè "falta una pàgina".
- Cada entrada ha de tenir una hipòtesi clara o s'elimina del backlog.
- Si una pàgina no canvia una decisió, no puja de prioritat.
- Una pàgina de feature amb impacte baix no passa per davant d'una home feble o d'una pàgina que resol una objecció crítica.
